VS Code snippets

2020-05-03
how-to
214 words 2 mins read

Contents

VS Code snippets are templates that make it easier to enter repeating code patters.

You can have both language specific code snippets and global snippets. To create your own snippets, go to File > Preferences > User Snippets.

Select a language, let’s say I want to create a snippet for markdown pages, so I will select markdown. Then add following code and save the file.

{
 "File Header": {
  "prefix": "header",
  "description": "Output a file header for a hugo blog post",
  "body": [
   "---",
   "title: $TM_FILENAME",
   "date: $CURRENT_YEAR-$CURRENT_MONTH-$CURRENT_DATE $CURRENT_HOUR:$CURRENT_MINUTE:$CURRENT_SECOND",
   "toc: true",
   "autoCollapseToc: true",
   "draft: false",
   "comment: true",
   "tags: []",
   "categories: []",
   "---",
  ]
 }
}

You might also in this case want to make some changes in your user settings (File > Preferences > Settings) to enable this newly added snippet.

Enable quick suggestions for markdown:

1
2
3

"[markdown]":  {
    "editor.quickSuggestions": true
  }

To make sure that snippets always shows up first in the suggestion list, add this to your settings file.

1

"editor.snippetSuggestions": "top"

To use your new snippet, simply create a new md (markdown) file and type header (this is the prefix keyword you defined in your snippet and it can be anything).
